The Theme - Dreamscape Enforcer (Surrealist)

    The Premise: You are a protagonist diving into the minds to destroy their nightmares.

    The Action: Fighting conceptual enemies using imaginative weapons. The text can be beautifully bizarre and abstract.

    Procedural Hook: Rooms and enemies are generated based on the "fears" of the host. 
    You might fight a manifestation of "Heights" in one room and "Public Speaking" (which deals sonic damage) in another.

    Phase 1 (Weeks 1-4):
    - Basic OOP concepts. 
    - Create classes for Room, Player, and Enemy. 
    - Implement a simple gameplay loop where the player can move between rooms and attack enemies. 
    - Enemies can also attack the player.

    Phase 2 (Weeks 5-8): 
    - Polymorphism and abstraction. 
    - Multiple enemy types using inheritance. 
    - Abstract base class for all entities. 
    - File I/O to save and load game state.

    Phase 3 (Weeks 9-12): 
    - Procedural generation using STL (std::map for room graph, std::vector for encounter lists). 
    - Random room/encounter generation. 
    - Performance-aware containers. 
    - Memory management (smart pointers).
